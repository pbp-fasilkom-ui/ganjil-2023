# &lt;h1&gt; - Heading
Elemen heading adalah elemen teks berukuran besar yang dapat digunakan sebagai title atau awalan dari section pada website. Elemen heading dapat didefinisikan dengan menggunakan tag `<h1></h1>`.

Selain `<h1></h1>`, terdapat pula variasi-variasi lain dari heading, dimana angka yang lebih besar menandakan heading yang berukuran lebih kecil.

**Ukuran-ukuran heading yang valid:** `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`
