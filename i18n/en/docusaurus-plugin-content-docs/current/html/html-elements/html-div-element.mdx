# &lt;div&gt; - Container
Elemen div atau container yang dapat digunakan untuk menggabungkan beberapa elemen kedalam satu elemen gabungan.

Elemen ini **sangat berguna** dalam mengimplementasikan layout yang lebih kompleks karena memungkinkan kita mengaplikasikan CSS style untuk positioning dan penampilan ke masing-masing kelompok komponen.

Tanpa menggunakan container, akan **sangat sulit** atau bahkan **tidak mungkin** untuk mengatur posisi sekelompok elemen pada halaman dengan rapih, sehingga elemen ini adalah elemen yang fundamental. 

### Beberapa Fungsi Container
- Mempermudah manajemen elemen pada sebuah page yang kompleks.
- Memungkinkan kita membagi sebuah halaman menjadi beberapa bagian terpisah.
- Bersama dengan CSS dapat digunakan untuk memposisikan sekelompok elemen relatif terhadap halaman (misal ke bagian tengah halaman).
