# &lt;a&gt; - Anchor

Anchor element is an element to create a link to another website on the Internet.
It is also can be used to create a link to another element in the same website.
The element can be defined by using a pair of `<a>` and `</a>` tag

Link yang ingin dituju oleh elemen anchor dapat didefinisikan dengan memberikan atribut href ke anchor tag yang nilainya berupa string URL.

#### Daftar Atribut Anchor Tag:
- `href`: URL yang ingin dituju
- `target`: Target window dimana halaman akan dibuka. Dapat memiliki nilai sebagai berikut:
  - `_self`: Buka link pada halaman yang sama (default).
  - `_blank`: Buka link pada halaman baru.
  - `_parent`: Buka link pada halaman *parent* (khusus untuk iframe).
