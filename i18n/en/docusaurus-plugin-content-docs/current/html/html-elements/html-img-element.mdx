# &lt;img&gt; - Image
Elemen image adalah elemen yang dapat digunakan untuk menampilkan gambar di sebuah halaman web. Elemen image dapat didefinisikan dengan menggunakan tag `<img />`.

Untuk memberi tahu browser image yang harus ditampilkan, kalian harus mendefinisikan value dari atribut `src` pada tag image. Selain itu terdapat pula beberapa atribut lain yang mengatur properti dari gambar seperti yang dituliskan pada bagian selanjutnya.

### Daftar Atribut Image Tag
- `src`: URL yang ingin dituju
- `alt`: Deskripsi dari image yang ditampilkan (digunakan ketika image tidak dapat di-load dan untuk accessibility)
- `height`: Set height dari image yang ingin ditampilkan (sebaiknya menggunakan CSS saja)
- `width`: Set width dati image yang ingin ditampilkan (sebaiknya menggunakan CSS saja)
