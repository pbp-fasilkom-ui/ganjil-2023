# &lt;p&gt; - Paragraph

The paragraph element is an element used for containing a text value. It can be
defined by using a pair of `<p>` and `</p>` tag.

Any text contained in the paragraph element is, by default, will be displayed
similar to a paragraph in a written document. The text will be contained in a
block and have a small margin at the bottom of the block to give small distance
to the next paragraph.
